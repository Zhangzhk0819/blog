coding
  • ceph

简介

Ceph 是一个分布式文件存储系统，其工作原理可以参考官方文档。本博客的主要目的是搭建 ceph 分布式集群，实现多个服务器的存储共享，这在 python 的多进程运行中非常方便。这样一来，我们只需要使用 pycharm 与其中一台服务器同步文件，其他所有服务器都能同时访问并运行。
